# finance-kyc-training（培训项目）

这是一个用于学习金融 KYC 相关系统的前端练习项目。
已经为你准备好一键运行脚本 —— 不需要写代码也能跑起来！

------------------------------------------------
【1️⃣ 运行前需要准备什么？】
------------------------------------------------

✔ Windows 电脑（必须）

因为项目提供的是 run.bat 脚本，只支持 Windows。

✔ 安装 Node.js（非常重要）

1）打开：https://nodejs.org
2）下载 “LTS 推荐版”（绿色按钮）
3）一路下一步安装即可

安装完成后，不需要额外设置。

------------------------------------------------
【2️⃣ 项目结构说明】
------------------------------------------------

你会看到类似这样的文件：

finance-kyc-training
 ├─ src/              —— 页面和组件
 ├─ index.html        —— 入口页面
 ├─ package.json      —— 依赖与脚本配置
 ├─ vite.config.ts    —— 运行配置
 ├─ tsconfig.json     —— TypeScript 配置
 ├─ run.bat           —— ★ 一键启动脚本
 └─ ...

不用全部看懂，
你只需要记住一句话：

👉 运行项目 = 双击 run.bat

------------------------------------------------
【3️⃣ 如何启动项目？】
------------------------------------------------

🟡 第一步：解压项目（如果是压缩包）

解压后进入：

finance-kyc-training

🟡 第二步：运行

找到文件：

run.bat

👉 直接双击运行（等几分钟）

------------------------------------------------
【4️⃣ 运行过程中会发生什么？】
------------------------------------------------

第一次运行时会自动：

✔ 安装项目需要的依赖
✔ 启动开发服务器
✔ 自动打开浏览器页面

浏览器会出现类似：

http://localhost:5173

看到系统界面 = 成功 🎉

⚠️ 第一次可能会比较慢，请耐心等待。

------------------------------------------------
【5️⃣ 常见问题】
------------------------------------------------

❌ 双击 run.bat 没反应
👉 右键 → 以管理员身份运行

-----------------------------------

❌ 浏览器没有自动打开
👉 手动输入：

http://localhost:5173

-----------------------------------

❌ 黑框一闪而过 / 报错
👉 99% 是 Node.js 没安装好

重新安装：https://nodejs.org
